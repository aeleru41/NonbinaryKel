# Nonbinary Kel by aeleru41

Leah was the first character I romanced in Stardew Valley, and I thought it was so cool that they made an explicitly lesbian character -- then I found out that my (then) male friends playing with me had no idea. Since Leah being queer was so important to me, I want everyone to be able to have this experience by making Kel nonbinary. Now you can enjoy a much queerer Leah and choose from a selection of sprites for them!

Anyway, this is a simple mod that changes dialogue about Kel to be gender neutral and adds ability to select which sprite they use. Available here or on [NexusMods](https://www.nexusmods.com/stardewvalley/mods/43415).

## Getting Started

### Dependencies

* [SMAPI 4.5.1+](https://smapi.io/)
* [Content Patcher 2.9.0+](https://www.nexusmods.com/stardewvalley/mods/1915)
* (Optional) [GMCM 1.16+](https://www.nexusmods.com/stardewvalley/mods/5098) can be used to easily configure the mod.

### Installing

* Download the latest release [here](https://github.com/aeleru41/NonbinaryKel/releases/latest).
* Unzip folder and extract contents to your mods folder.

### Configuration
* Run Stardew once to generate a config.json file.
* If GMCM is installed, go to the settings cog and select the Nonbinary Kel mod. Choose a sprite :3
* Otherwise, close Stardew Valley. Return to the mods folder where you installed the mod. Go into the [CP] Nonbinary Kel folder and open config.json.
* Change the text following "KelSprite" to "Androgynous", "Male", or "Female" to choose your sprite!

## Compatibility
This mod will conflict with any mod that edits the `LeahsExMale.png` and `LeahsExFemale.png` sprites in the characters section. It will also conflict with mods that change her 4 heart event, the internet version of her 8 heart event, and her 10 heart event. Please let me know about any conflicts and I will endeavour to make them compatible!
